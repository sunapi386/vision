# Chapter 1: The Shift

Every decade, a structural change rewires work. The internet collapsed distribution costs. Cloud computing made infrastructure variable. Mobile put a computer in every pocket. Each shift created new monopolies, new opportunities, and new dependencies.

AI is the next one — but it's different in a way that matters. Previous shifts changed WHERE work happens or HOW MUCH infrastructure costs. AI changes WHO does the work. Tasks that required human specialists — contract review, data analysis, clinical documentation, code generation — can now be performed by software. Not hypothetically. Today.

This creates a profound asymmetry. The companies building AI capture compounding value — every interaction improves their models, deepens dependency, and generates revenue. The companies using AI rent capability month-to-month, accumulating invoices instead of assets. But history tells us this asymmetry is temporary. Railroads, electricity, telephony, cloud computing — every infrastructure monopoly eventually commoditized. The question is who builds the layer that enables the transition.

## Sections
